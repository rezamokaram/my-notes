
# ‌ Installation

## ‌ Commands

Installing argo using argo helm repo:

```sh

# creating a namespace for argocd
kubectl create namespace argocd

# add argo repo
helm repo add argo https://argoproj.github.io/argo-helm

# install
helm install argocd -n argocd argo/argo-cd

# for using argocd in browser http://localhost:8080
kubectl port-forward service/argocd-server -n argocd 8080:443

# for login to argocd ui
# username is admin
# you can find password here
kubectl -n argocd get secret argocd-initial-admin-secret -o jsonpath="{.data.password}" | base64 -d
# e.g i2dOsPrjnZYoy6mv

```

## ingress controller

for using ingress controller and do not port forwarding for argocd:

```sh
#‌ installing ingress controller
kubectl apply -f https://raw.githubusercontent.com/kubernetes/ingress-nginx/master/deploy/static/provider/kind/deploy.yaml

# set argo ingress to use nginx
helm upgrade argocd --set configs.params."server\.insecure"=true --set server.ingress.enable=true --set server.ingress.ingressClassName="nginx" -n argocd argo/argo-cd
```

for installing cli:

```sh
# Detect OS
ARGO_OS="darwin"
if [[ "$(uname -s)" != "Darwin" ]]; then
  ARGO_OS="linux"
fi

# Download the binary
curl -sLO "https://github.com/argoproj/argo-workflows/releases/download/v3.7.1/argo-$ARGO_OS-amd64.gz"

# Unzip
gunzip "argo-$ARGO_OS-amd64.gz"

# Make binary executable
chmod +x "argo-$ARGO_OS-amd64"

# Move binary to path
mv "./argo-$ARGO_OS-amd64" /usr/local/bin/argo

# Test installation
argo version
```
